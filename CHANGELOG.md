# Changelog

## v2.0
- Added a results page shown after finishing a quiz: score, percentage, and a short performance message
- Added a stats dashboard: total quizzes completed, average score, strongest subject, weakest subject
- Added an export feature — download completed quiz results as a `.txt` file
- Added two new question types (True/False, Short Answer) alongside Multiple Choice, with a selector for mixed or single-type quizzes
- Short-answer questions show a model answer for self-grading, since free-text isn't auto-gradable

## v1.0
- Initial release: paste notes or a topic, choose subject/difficulty/question count, generate a multiple-choice quiz
- Answer selection with instant right/wrong feedback and explanations
- Persistent quiz history and a "questions you've missed" list, saved locally in the browser
- Deployed as a static site on GitHub Pages
